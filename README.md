# UniversalPolycentricModel

Веб-приложение для применения полицентрической модели сельской агломерации к данным выбранного региона.

Приложение разработано в рамках выпускной квалификационной работы по теме:

**«Анализ мотивации проживания на сельских территориях методами машинного обучения»**

## Описание проекта

Данный репозиторий содержит универсальную версию приложения для построения полицентрической модели сельской агломерации.

В отличие от исследовательской версии, где сравнивались разные алгоритмы кластеризации, в данном приложении используется уже выбранная модель — **Spectral Clustering**. Подбор гиперпараметров сохраняется и выполняется с помощью библиотеки **Optuna**.

Приложение может применяться к данным разных регионов при условии, что входные файлы имеют необходимую структуру.

## Входные данные

Для работы приложения необходимо загрузить три файла:

* файл с численностью населения территорий;
* файл с объектами инфраструктуры, рейтингами и количеством отзывов;
* матрицу расстояний между территориями.

На основе этих данных рассчитываются показатели обеспеченности, качества инфраструктуры и взаимодополняемости территорий.

## Методика расчёта

В приложении выполняются следующие этапы:

1. Загрузка и сопоставление данных по территориям.
2. Расчёт показателей обеспеченности инфраструктурой.
3. Расчёт сглаженных оценок качества объектов.
4. Определение сильных сторон и инфраструктурных дефицитов территорий.
5. Формирование матрицы полицентрической связи.
6. Подбор гиперпараметров с помощью Optuna.
7. Кластеризация территорий методом Spectral Clustering.
8. Формирование итоговых групп, ролей территорий и таблиц результатов.

## Используемая модель

В универсальной версии используется метод:

* Spectral Clustering.

Подбираемые параметры:

* коэффициент сглаживания рейтингов;
* порог расстояния между территориями;
* число ближайших соседей;
* предельное расстояние связи;
* параметр пространственного затухания;
* количество кластеров;
* способ назначения меток.

## Проверка на данных Кировской области

Универсальная версия приложения была проверена на данных Кировской области. Это позволяет показать, что выбранный подход может быть применён не только к исходному региону исследования, но и к другой территории при наличии аналогичных входных данных.

## Состав репозитория

```text
app.py                          # Streamlit-приложение
model_pipeline_universal.py     # расчётная часть универсальной модели
requirements.txt                # зависимости проекта
```

При необходимости в репозиторий также могут быть добавлены тестовые входные файлы:

```text
population_kirov.xlsx
results_kirov.xlsx
distances_matrix_kirov.xlsx
```

## Запуск приложения

Для локального запуска необходимо установить зависимости:

```bash
pip install -r requirements.txt
```

Затем запустить приложение:

```bash
streamlit run app.py
```

## Назначение в рамках ВКР

Данная версия приложения используется для демонстрации применения выбранной модели к данным другого региона. Исследовательская версия приложения применялась для сравнения нескольких моделей на данных Нижегородской области, после чего в качестве основной была выбрана Spectral Clustering. Универсальная версия использует этот подход для дальнейшего расчёта полицентрической структуры территорий.
